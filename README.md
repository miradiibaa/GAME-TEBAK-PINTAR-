# 🎮 Tebak Pintar

Game edukasi berbasis browser dengan tema pixel art retro. Terdiri dari 3 level yang menguji kemampuan logika dan matematika dasar.
---

## 🕹️ Level

**Level 01 — Ganjil atau Genap**  
Diberikan 5 angka acak antara 1–50. Pilih apakah angka tersebut ganjil atau genap.

**Level 02 — Operasi Matematika**  
Selesaikan 4 soal matematika dasar secara berurutan: penjumlahan, pengurangan, perkalian, dan pembagian. Ketik jawaban dan tekan Enter atau tombol JAWAB.

**Level 03 — Tebak Angka**  
Tebak angka rahasia antara 1 hingga 10. Program memberi petunjuk apakah tebakanmu terlalu besar atau terlalu kecil. Kamu punya 5 kesempatan.

---

## ✨ Fitur

- Tampilan pixel art retro dengan font Press Start 2P
- HUD dengan bar HP yang berkurang setiap soal
- Sistem bintang (1–3) sebagai penilaian di setiap level
- Log riwayat tebakan di Level 3
- Mendukung input keyboard (Enter untuk menjawab)
- Tidak membutuhkan instalasi atau server — cukup buka di browser
